I cannot see my property in the list

The list of properties you can select depends on whether you want an agent to carry out either a check, a challenge or both. A maximum of two agents can assigned to a property, with permissions to either check or challenge a property valuation.

  • If you are assigning an agent with both check and challenge rights, then a list of properties that have no agents already assigned will be displayed.
  • If you are assigning an agent with check rights only, then a list of properties with no agent assigned and properties where an agent has already been assigned with challenge rights will be displayed.
  • If you are assigning an agent with challenge rights only, then a list of properties with no agent assigned and properties where an agent has already been assigned with check rights will be displayed.
