You’ll need to be an administrator for the Government Gateway account to add or remove another person.

You’re an administrator if you created the Government Gateway account or if you have the ‘Manage Administrators’ and ‘Manage Assistants’ options (shown on the left hand side after signing in to Government Gateway and clicking on ‘Your account’).

Add another person

To add another person you’ll need to:

  • sign in to Government Gateway
  • select ‘Manage Administrators’ (or Manage Assistants’)
  • select ‘Add an Administrator’ (or ‘Add an Assistant’)
  • enter the details of the new user and create a password
  • submit the details
  • make a note of the User ID

You can then give the User ID and password to the new user so that they can register for the service.

Remove another person

To remove another person you’ll need to:

  • sign in to Government Gateway
  • select ‘Manage Administrators’ (or Manage Assistants’)
  • click on ‘delete’ next to the person that you want to remove
  • confirm that you want to delete the person you have selected

You’ll then be asked to sign in again for security reasons. The person will now no longer display on your ‘Manage Administrators’ or ‘Manage Assistants’ screens.
