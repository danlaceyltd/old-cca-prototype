Rating list

The VOA calculates the rateable value for each non-domestic property in England and Wales and produces a new rating list usually every 5 years. The list includes the address of the property, a description of the property, its rateable value and the date that value has effect from. The 2017 rating list came into effect on 1 April 2017.

Close

Date applied

This is the date that the physical circumstances affecting the rateable value take effect from. For example, if an area was added to the property on 1 January 2018, there should be a revised valuation with effect from that date.

If there have been no physical changes to the property since the beginning of the rating list, this will be 1 April 2017.

Close

Valuation scheme reference

The VOA groups similar properties in the same area together into a ‘valuation scheme’ so that they value them fairly and consistently. These schemes are drawn up using rental information that the VOA collects. All properties in the same scheme will be given a basic value from the same value range. Individual characteristics of properties can be reflected by adjusting the basic rate.

For example, in a valuation scheme, the VOA may assume that there is no fire protection. If one of the properties in that scheme has it, they may apply a higher value to that property because of this feature. The final value for a property will depend on an adjustment of all of the individual attributes and factors together.

Each scheme has a unique reference number and describes which type of property it covers.

Close

Part domestic (composite) property

If a hereditament (a unit of property with the same address and local authority reference number) has a mix of domestic and non-domestic uses (such as a shop with the shop owner’s flat above) it may be described as ‘part domestic (composite)’. This means it will have both a non-domestic valuation and a Council Tax band. Both uses will be considered in the valuation.

Close

Basis of measurement

The basis of measuring that the VOA have used for this property. A property is always assumed to be vacant and to let and most tenants’ fittings and furniture are ignored when measuring. The basis will usually be Net Internal Area (NIA) for offices and shops, and Gross Internal Area (GIA) for warehouses and industrial properties.

NIA is the total usable area of each floor within a building, measured to the inside face of the boundary walls. It doesn’t include areas such as internal structural walls and stairwells.

GIA is the total area of a building measured to the internal face of the external walls at each floor level, including internal walls and stairwells.

Close

Date of last list alteration

This is the date the VOA made the changes to the rating list. For example, if the VOA agreed with information received in a check relating to an effective date of 1 April 2017, the effective date would still be 1 April 2017 and the date of the list alteration would be the date the VOA updated the list.

Close

Relativity

Areas in a property are given a value in relation to the main space for the particular use . So, if the front of a shop (zone A) has a value of 1.0, other areas of the property will be valued against this, depending on where they are in the shop or whether the space is secondary to anyone occupying it for the same use.

Close

Local authority reference

This is a reference code assigned to the property by the local authority, which is used by both the council and the VOA. This is assigned to avoid confusion and to ensure that the rateable value used when the council calculates your business rates bill is for the right property.

Close

Compounded adjustments

Where there is more than one adjustment the values of each adjustment are multiplied. For example, an upward adjustment can be multiplied with a downward adjustment to reach a compounded value.

Fair maintainable trade

An estimate of the annual trade (excluding VAT) that a pub would be expected to achieve if operated in a reasonably efficient way. This is usually based on actual receipts.
