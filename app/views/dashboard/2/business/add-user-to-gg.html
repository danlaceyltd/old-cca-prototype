You’ll need to be an administrator for the Government Gateway account to add users. Youre an administrator if you created the Government Gateway account or if you have the ‘Manage Administrators’ and ‘Manage Assistants’ options (shown on the left hand side after signing in to Government Gateway and clicking on ‘Your account’).

To add a user, you’ll need to:

  • sign in to Government Gateway
  • select ‘Manage Administrators’ (or Manage Assistants’)
  • select ‘Add an Administrator’ (or ‘Add an Assistant’)
  • enter the details of the new user and create a password
  • submit the details
  • make a note of the User ID

You can then give the User ID and password to the new user so that they can register for the service.
